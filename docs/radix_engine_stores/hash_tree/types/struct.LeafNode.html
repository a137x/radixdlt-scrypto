pub fn leaf_hash(&self) -> Hash

Gets the leaf’s hash (not to be confused with a value_hash()). This hash incorporates the node’s key and the value’s hash, in order to capture certain changes within a sparse merkle tree (consider 2 trees, both containing a single element with the same value, but stored under different keys - we want their root hashes to differ).
